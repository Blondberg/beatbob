## Getting started

### Prerequisites

- python (Beatbob is written in Python, so will be hard without it) </br>
  > [Python downloads page](https://www.python.org/downloads/)

- pip
  ```sh
  python get-pip.py
  ```

- pipenv (optional, for managing requirements)
  ```sh
  pip install -U pipenv
  ```
